# omnilink-solutions

Static, Vercel-ready build for **OmniLink Solutions** with:
- Service mega-grid + chat lead bot
- Quick lead form → local dashboard
- Contractor signup, login, and dashboard
- All data stored locally (no backend required). Replace with your API later.

> Built 2025-11-09

## Files
- `index.html` — main page with categories and services (tap-to-chat).
- `contractor.html` — contractor application form.
- `contractor-login.html` — admin login (demo auth).
- `contractor-dashboard.html` — view leads and applications.
- `review.html` — demo reviews.
- `assets/app.css` — styles.
- `assets/app.js` — shared JS, chat, dashboard.
- `assets/services.js` — master service catalog.
- `vercel.json` — safe headers and clean URLs.

## Deploy
Drag-and-drop this folder into Vercel, or push to GitHub and connect. Static only.
